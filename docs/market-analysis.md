# Анализ рынка систем автоматизации академического секретаря/методиста

> **Issue:** [#80](https://github.com/inf-sys-secretary-methodologist/inf-sys-secretary-methodist/issues/80)
> **Дата:** 2026-05-19
> **Версия проекта:** 0.153.13
> **Контекст:** дипломный проект, защита близко. Цель — обоснование позиционирования собственной системы относительно существующих российских и зарубежных решений.
> **Методология:** 11 поисковых запросов + 20+ прямых fetch-чтений vendor-страниц (см. раздел [Источники](#источники-и-ссылки)).
>
> **Известные verification gaps:**
> - SITITO 2026 academic peer-reviewed paper "Сравнительный анализ информационных систем для автоматизации управления учебным процессом в вузе" — таймаут при fetch, прочитан только title из search results.
> - Реестр Минцифры (reestr.digital.gov.ru) — проверка реестровых номеров конкретных систем не выполнена; статус ✅ указан на основе vendor-страниц где это явно декларируется.
> - Pricing для ТАНДЕМ / Галактика / Naumen / ММИС / Modeus — все используют custom quote model и публично не раскрывают; цифры являются обобщёнными по аналогии с 1С-моделью.

## Executive summary

На российском рынке доминируют **семь основных систем**:

| Класс | Системы | Целевая | Pricing pattern |
|-------|---------|---------|-----------------|
| **СПО-первичные** | 1С:Колледж, 1С:Колледж ПРОФ | колледжи, техникумы | Разовая лицензия 88,400-179,300 ₽ + ИТС |
| **ВУЗ-первичные классические** | 1С:Университет ПРОФ, ТАНДЕМ.Университет, Галактика ВУЗ, Naumen University, ММИС Деканат | вузы (с поддержкой СПО) | Разовая 282,700 ₽ или custom quote |
| **ВУЗ-нового поколения** | Modeus (CUSTIS) | топовые исследовательские вузы | Enterprise consortium model |

За рубежом — **Moodle/Sakai** (open-source, бесплатно) vs **Canvas/PowerSchool/Blackboard** (per-institution SaaS subscription $50k-200k+/год, обычно $5-30/student/year, см. раздел 2.2).

**Ключевой gap, который закрывает наша система:**
- Малые и средние СПО/частные образовательные учреждения не могут позволить enterprise-внедрения 1С:Колледж ПРОФ (179,300 ₽ + 30-60% внедрение + 68,400 ₽/год ИТС ПРОФ)
- Moodle/Sakai не покрывают российские специфики (1С:Бухгалтерия интеграция, ФГОС, отчётность для деканата, ФИС ГИА/ФРДО)
- Современный UX (React + WebSocket realtime + mobile-first + PWA) у большинства российских решений отсутствует — они построены на legacy desktop-frame UI (1С:Тонкий клиент или J2EE толстый клиент)

**Наше позиционирование:** modern web-stack (Go + Next.js) + integration-ready (1С OData + n8n + Telegram + Composio) + СПО-первичный фокус + open-source self-hosted для образовательных учреждений с ограниченным бюджетом.

**Слабые стороны**:
- Desktop-frame UI (через 1С:Тонкий клиент), не mobile-friendly
- Внедрение требует 1С-партнёра (доплата 30-60% к лицензии)

- Нет современного REST API / WebSocket realtime
- Ежегодные подписки на обновления

## 3. Сравнение бизнес-моделей

### 3.1 Российский рынок — разовая лицензия + ITS субподписка

**Модель "Разовая лицензия + ИТС"** (1С):
- Разовая покупка: 88,400-282,700 ₽
- Ежегодные обновления конфигурации: 31,200 ₽ (12 мес)
- Подписка ИТС ПРОФ: 68,400 ₽/год
- Внедрение через партнёров: +30-60% к лицензии
- Customer lifecycle cost ~ **3-4x лицензии за 5 лет**

**Модель "Custom quote per project"** (Тандем, Галактика, Naumen, ММИС, Modeus):
- Прозрачные публичные цены отсутствуют (sales-driven)
- Учитывается размер учреждения, набор модулей, срок лицензии
- Внедрение часто включено в quote
- 12-месячная тех. поддержка обычно в стоимости
- После warranty — renewal или upgrade

### 3.2 Зарубежный рынок — SaaS subscription доминирует

- **Per-student pricing**: $5-30/student/year (typical higher ed, см. раздел 2.2 verified)
- **Institutional contracts**: $50,000-200,000+/год для колледжа на 5-10k студентов (Canvas verified)
- Tiered (Basic / Pro / Enterprise) с разным набором features (core LMS vs advanced reporting + dedicated support)
- Per-institution negotiation: factors — user count, FTE enrollment, contract length, add-ons
- Open-source path для бюджетных учреждений (Moodle/Sakai self-hosted)
- Hosted-by-affiliates модель (Sakai через Longsight/Learning Experiences/EDF)

### 3.3 Антипаттерны (что не работает)

- **Freemium для гос. учреждений**: не подходит из-за закупочных процедур 44-ФЗ (нужен реестр Минцифры + договор)
- **Чисто SaaS без on-premise** опции для гос. вузов: требования по хранению ПДн на территории РФ
- **Mobile-first без desktop**: основная пользовательская база (методисты, секретари, преподаватели) работает с desktop рабочих мест
- **AI-only** позиционирование: бюджетный сектор пока не покупает "AI-powered" премию, нужна базовая функциональность с AI как opt-in

### 7.2 TAM/SAM/SOM расчёт

> **Disclaimer**: все цифры ниже — top-down оценки на основе публичной статистики Минпросвещения и средней TCO 1С:Колледж. Для investor-grade анализа требуются прямые опросы вендоров о размере их клиентской базы.

**TAM (Total Addressable Market)** — все учреждения СПО в РФ (3-летний горизонт):
- 3,200 колледжей × ~680k ₽ TCO 1С:Колледж за 3 года = **₽2.18 млрд / 3 года** или **~₽725M/год**
- Если включить 393 вуза × ~1M ₽ за 3 года (282k лицензия + 3×ИТС эст. + внедрение) = +₽393M / 3 года = **~₽130M/год**
- **Total TAM ≈ ₽855M/год** для российского рынка специализированного ПО автоматизации образования

**SAM (Serviceable Available Market)** — учреждения с профилем подходящим под нашу систему:

Точная доля частных СПО в РФ варьируется по источникам (Росстат vs Минпросвещения публикуют разные срезы), но по обобщённым оценкам составляет **15-20%** = **480-640 учреждений**. Для них реестр Минцифры не критичен (нет требований 44-ФЗ).

Принимаем для расчёта **средние 18% = 580 учреждений**:
- Если каждое тратит хотя бы 1/3 от TCO 1С:Колледж = ~230k ₽/год на специализированный софт → **SAM ≈ ₽133M/год**
- Этот SAM включает: replacement цикл для тех кто уже на 1С/Excel + greenfield малые частные
- Расширение: государственные средние/малые СПО (далеко от Москвы, где бюджеты скромнее) могут добавить ещё ~₽50M/год

**Итоговая SAM-оценка**: **₽130-200M/год** для нашего профиля.

**SOM (Serviceable Obtainable Market)** — реалистичная захватываемая доля при заходе как новичок:
- 1-3 pilot deployments в первый год после защиты + закрытие реестра Минцифры (~6 мес)
- Каждый pilot: ~50-150k ₽ внедрение + free open-source core = **₽150-450k revenue в первый год**
- Органический рост через сарафанное радио + отраслевые конференции (CISO Forum, СПО-2027)
- 3-летняя цель: **5-10 paying customers** × ~120k ₽/год support+upgrades = **₽600k-1.2M/год** к концу 3-го года
- Это **<1% SAM** — реалистично для bootstrap'ed product без enterprise sales force

### 7.3 Конкурентная ситуация в SAM

В сегменте **малых/средних частных СПО (наш SAM ≈ 580 учреждений)**:
- **1С:Колледж** — основной существующий конкурент; высокая цена для частных СПО (lifetime ~680-765k за 3 года)
- **Modeus** (CUSTIS) — не targeting (top-tier ВУЗ-focused, enterprise consortium model)
- **Галактика / Naumen / ТАНДЕМ / ММИС** — enterprise custom quote'ы недосягаемы для малых частных
- **Moodle / Sakai** — не покрывают workflow деканата (нет приёмной кампании, контингента, ФГОС-отчётности)

**Гипотеза о текущем стеке частных СПО** (не подтверждено опросом, требует pilot-валидации): большинство используют комбинацию Excel/Google Sheets + 1С:Бухгалтерия + WhatsApp/Telegram для коммуникации + локальные решения для расписания. Полностью интегрированных систем с современным UX нет, что создаёт окно возможности.

## 8. Выводы и рекомендации для защиты

### 8.1 Основные тезисы для защиты

1. **Главный конкурент** — **1С:Колледж ПРОФ** (179,300 ₽ + 30-60% внедрение + 68,400 ₽/год ИТС = ~677k ₽ за 3 года минимум; **1,150+ внедрений за историю продукта** при разработке ред. 2.1). Наша система предлагает альтернативу с **современным web-stack и без vendor lock-in**.

2. **Рынок есть**: **3,200 колледжей и техникумов в РФ** + 393 вуза с СПО, **3.9M студентов** (50-летний рекорд по приёму в 2025). Наш SAM ≈ 800 малых/средних частных СПО (~₽160M/год потенциал).

3. **Самое слабое место в позиционировании** — отсутствие в реестре Минцифры. Подача в реестр требует ~2-3 месяца и формального юр.лица. Возможный followup после защиты диплома.

4. **Самое сильное место** — **5-6 уникальных features**, которых **нет ни у одного российского конкурента в одном пакете** (по результатам проверки 7 ключевых систем — см. матрица в разделе 6):
   - AI RAG-чат с pgvector (никто из 7 проверенных систем не предлагает)
   - Realtime WebSocket мессенджер (никто из 7)
   - Telegram bot integration через Composio (никто из 7)
   - i18n × 4 ru/en/fr/ar с RTL (никто из 7)
   - Open-source self-hosted (никто из 7 enterprise-конкурентов)

   *Mobile* / MFA / WebPush имеются у части конкурентов (Modeus modern web, Naumen cert auth и т.д.) — не считаем уникальными.

5. **Рыночная стратегия** — pilot deployments в малом/среднем частном СПО с командой 1-2 ИТ-специалистов и бюджетом ≤ 200k ₽/год на ИТ-инфраструктуру.

### 8.2 Pricing recommendation для будущей коммерциализации

Двухуровневая модель (open-source core + платные сервисы):

**Tier 1 — Open-source self-hosted core** (free):
- Полный функционал, MIT/Apache license
- Цель: привлечение пользователей, формирование сообщества, retention через сетевой эффект
- Требует от учреждения: 1-2 ИТ-специалистов, Linux VM (~10-30k ₽/мес VPS hosting), базовый sysadmin

**Tier 2 — Профессиональные сервисы** (paid):
- **Внедрение под ключ**: 50-150k ₽ (single-instance setup + migration from Excel/legacy + 1С OData configuration + initial staff training)
- **Ongoing support contract**: 5-15k ₽/мес (priority bug fixes + security patches + version upgrade assistance)
- **Custom development**: 3-5k ₽/час (новые модули, кастомные интеграции)

**Альтернативная SaaS-модель** (для тех кто не хочет self-hosting):
- 99 ₽/студент/месяц с минимумом 500 студентов = 49,500 ₽/мес = **594,000 ₽/год** для колледжа на 500 студентов
- Для 1000 студентов = **1,188,000 ₽/год**
- За 3 года: 1.78M (500 студ.) или 3.56M (1000 студ.)

**Сравнение TCO за 3 года** (на 1000 студентов, **fair-compare** — все опции с hosting):

| Опция | Лицензия | Внедрение | Operating (3 года) | **TCO за 3 года** | Преимущества |
|-------|----------|-----------|---------------------|--------------------|--------------|
