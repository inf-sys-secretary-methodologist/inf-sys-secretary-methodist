## Поток привязки Telegram

```
┌──────────────────┐                              ┌──────────────────┐
│   Веб-интерфейс  │                              │   Telegram Bot   │
└────────┬─────────┘                              └────────┬─────────┘
         │                                                 │
         │  1. Click "Подключить Telegram"                 │
         │                                                 │
         │  2. POST /api/telegram/verification-code        │
         │     ─────────────────────────────────────►      │
         │                                                 │
         │     ◄─────────────────────────────────────      │
         │     {code: "a1b2c3d4", bot_link, expires_at}    │
         │                                                 │
         │  3. Показать код + QR/ссылку                    │
         │     https://t.me/bot?start=a1b2c3d4             │
         │                                                 │
         │                                                 │  4. User clicks link
         │                                                 │     or sends code
         │                                                 │
         │                                                 │  5. /start a1b2c3d4
         │                                                 │     ────────────────►
         │                                                 │
         │                       ┌─────────────────────────┼─────────────────────┐
         │                       │   TelegramWebhookHandler│                     │
         │                       │                         │                     │
         │                       │  6. Parse message       │                     │
         │                       │  7. Extract code        │                     │
         │                       │  8. Call VerifyCode()   │                     │
         │                       │                         │                     │
         │                       │  ┌─────────────────────────────────────────┐  │
         │                       │  │ TelegramVerificationService.VerifyCode()│  │
         │                       │  │                                         │  │
         │                       │  │  a) Check code exists & valid           │  │
         │                       │  │  b) Check not expired (15 min)          │  │
         │                       │  │  c) Check chat_id not used              │  │
         │                       │  │  d) Create TelegramConnection           │  │
         │                       │  │  e) Mark code as used                   │  │
         │                       │  │  f) Enable telegram in preferences      │  │
         │                       │  │  g) Send welcome message                │  │
         │                       │  └─────────────────────────────────────────┘  │
         │                       └───────────────────────────────────────────────┘
         │                                                 │
         │                                                 │  9. "Аккаунт привязан!"
         │                                                 │     ◄────────────────
         │                                                 │

         │      ─────────────────────────────────────►     │

         │      {connected: true, username: "..."}         │
         │                                                 │

         │                                                 │
```
