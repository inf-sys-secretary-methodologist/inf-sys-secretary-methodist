# Design System - Информационная система секретаря-методиста

Добро пожаловать в документацию компонентов проекта.

## О проекте

Информационная система для секретаря-методиста образовательного учреждения.
Построена на современном стеке технологий с использованием лучших практик UI/UX.

## Технологический стек

- **Next.js 15** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS 4** - стилизация
- **shadcn/ui** - базовые компоненты
- **Magic UI** - анимационные эффекты
- **21st.dev** - дизайн-система
- **Recharts** - графики

## Структура компонентов

### UI Components (`UI/`)

Базовые компоненты интерфейса:

- **Button** - кнопки с различными вариантами
- **Input** - поля ввода
- **FloatingInput** - поля с плавающим label
- **Card** - карточки контента
- **Badge** - метки и теги
- **Tabs** - табы навигации

### Magic UI (`Magic UI/`)

Анимированные компоненты:

- **NumberTicker** - анимированные числа

### Navigation (`Navigation/`)

Навигационные компоненты:

- **NavBar** - "tubelight" навигация с эффектом свечения

### Dashboard (`Dashboard/`)

Компоненты дашборда:

- **StatsCard** - карточки статистики
- **TrendChart** - графики трендов
- **ActivityFeed** - лента активности

## Цветовая схема

### Статусы изменений

```css
/* Положительное */
bg-green-500/20 text-green-600 border-green-500/50

/* Отрицательное */
bg-red-500/20 text-red-600 border-red-500/50
```

### Стиль карточек

```css
/* Светлая тема */
bg-white border-gray-200

/* Темная тема */
bg-black/95 border-gray-700
```

## Паттерны использования

### Анимированное число

```tsx
<div className="text-3xl font-bold">
  <NumberTicker value={1234} />
</div>
```

## Навигация по документации

Используйте боковую панель для навигации по компонентам.
Каждый компонент имеет:

- **Docs** - документация с описанием
- **Canvas** - интерактивный предпросмотр
- **Controls** - настройка параметров

## Dark Mode

Используйте переключатель темы в toolbar для проверки
компонентов в темном режиме.
