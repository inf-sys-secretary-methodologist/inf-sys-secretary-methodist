# Public Directory

Эта директория содержит статические файлы, которые будут доступны напрямую через URL.

## Использование

Файлы в этой директории будут доступны по корневому пути:

- `public/logo.svg` → `/logo.svg`
- `public/images/hero.png` → `/images/hero.png`

## Рекомендации

1. **Favicon и метаданные**: Поместите `favicon.ico`, `robots.txt`, `sitemap.xml` здесь
2. **Изображения**: Храните статические изображения, которые не требуют оптимизации
3. **Шрифты**: Локальные шрифты (если не используете next/font)
4. **Манифесты**: `manifest.json` для PWA

## Примечания

- Файлы обслуживаются как есть, без обработки Next.js
- Для изображений рекомендуется использовать `next/image` с оптимизацией
- Не храните конфиденциальную информацию в этой директории
