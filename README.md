The project is a software system designed to assist in the management and methodological support of information systems. It aims to automate administrative and secretarial tasks, streamline workflow processes, and provide methodological guidance to users.

## Main features

- Document management
- Scheduling
- Reporting
- Communication tools to support efficient organizational operations

## Purpose

The project is created to:

- Enhance productivity
- Ensure consistency in process execution
- Facilitate information exchange within teams or departments
